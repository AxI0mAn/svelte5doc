Режим Разработки (Dev)	

npm run dev	

Самая важная команда. Запускает сервер разработки. Он наблюдает за вашими файлами (HTML, CSS, JS/Svelte) и автоматически пересобирает приложение и обновляет браузер при каждом сохранении (Hot Module Replacement, HMR).

---------------------------

Открыть Браузер	

npm run dev -- --open	

То же, что и npm run dev, но с добавлением флага --open. Флаг --open говорит SvelteKit (через Vite) автоматически открыть приложение в вашем браузере по умолчанию.

---------------------------

Сборка для Прода (Prod)	

npm run build	

Компилирует все ваши Svelte-компоненты в высокооптимизированный ванильный JavaScript, HTML и CSS. Создает папку build/ с готовыми файлами.

--------------------------

Просмотр Прода	

npm run preview	

Запускает локальный веб-сервер, который показывает сборку, созданную командой npm run build. Используется для проверки, как приложение будет работать в продакшене перед деплоем.
